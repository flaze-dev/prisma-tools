# prisma-tools
A cli tool that offers Prisma utility tools like exporting types.

### Install via `npi`
```bash
npm install -g @flaze/npi
```
```bash
npi install https://github.com/flaze-dev/prisma-tools.git
```

### Install Locally
```bash
$ git clone https://github.com/flaze-dev/prisma-tools.git
```
```bash
$ cd prisma-tools
$ npm install
$ npm run install:bin
```
